===============================

How to Play:
-Open index.html in your browser (JavaScript must be enabled. All modern browsers support the canvas element)
-Use the arrow keys to move in each direction.
-Avoid wandering bugs.
-Reaching the water earns you a point. Getting bugged resets your score.
